SVG 要素

path は、ここで紹介した単純なシェイプのほかに、より複雑な形状を描くときにも用います（地図の国境線など）。

SVG 要素のスタイル設定

  • fill（塗りつぶしの色）
  • stroke（外形線の色）
  • stroke-width（線の太さ）
  • opacity（透明度） : 0.0（完全に透明）から 1.0（完全に不透明）

レイヤーと描画の順序

SVG にはレイヤも深度の概念もありません。また、CSS の z-index プロパティもサポートしていません。
そのためすべてのシェイプは、1枚の xy 二次元平面上に配置されます。

fillやstrokeなど、色の指定する箇所ではrgba() を使用できる。
